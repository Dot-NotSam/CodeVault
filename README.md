# CodeVault

### A Java-Based Programming Knowledge & Snippet Manager for Students

---

## 📌 Overview

CodeVault is a console-based Java application designed to help students organize, store, and quickly retrieve important programming snippets and concepts.

While learning programming, students often struggle to remember syntax, algorithms, and reusable logic. CodeVault solves this by acting as a structured digital notebook where users can save, search, and manage their coding knowledge efficiently.

---

## 🎯 Problem Statement

Students frequently:

* Forget important code snippets and logic
* Struggle to organize learning materials
* Waste time searching for previously written code

This project addresses these challenges by providing a simple and structured system for managing programming knowledge.

---

## 💡 Features

* ➕ Add new code snippets with title, description, tags, and category
* 📂 Categorize snippets (DSA, OOP, File Handling, etc.)
* 🔍 Search snippets by keyword (title, tags, description, category)
* ⭐ Mark/unmark snippets as favorites
* 📋 View all stored snippets
* ❤️ View only favorite snippets
* 🕒 View recently added snippets
* 📊 View statistics (total snippets & category-wise count)
* ❌ Delete snippets
* 💾 Data persistence using file handling

---


## 🛠️ Technologies Used

* Java (Core Java)
* Object-Oriented Programming (OOP)
* Collections Framework (ArrayList, HashMap)
* File Handling (BufferedReader, BufferedWriter)

## ⚙️ How to Run

1. Clone the repository:

   ```
   git clone <your-repo-link>
   ```

2. Open the project in any Java IDE (IntelliJ / Eclipse / VS Code)

3. Compile and run:

   ```
   javac Main.java
   java Main
   ```

4. Use the menu to interact with the application
